# Dual-Weighted Naive Bayes (DW-NB)

Research implementation of **DW-NB**, a test-time posterior fusion method:

\[
\hat P_{DW}(c\mid x^*) \propto \hat P_{NB}(c\mid x^*)^{1-\lambda} \cdot W_c(x^*)^{\lambda}
\]

DW-NB keeps GaussianNB training unchanged and injects local WPRkNN-style correction (`w1`,`w2`,`w3`) only at prediction time.

## Project Layout

- `src/wprknn_weights.py`: Amer et al. Eq. 9-15 weighting components and composition.
- `src/lambda_selector.py`: fixed and inner-CV lambda strategies.
- `src/dw_nb.py`: `DWGaussianNB`, `DWGaussianNB_CV`, `NBkNNEnsemble`.
- `src/datasets.py`: 24 dataset loader + preprocessing/caching.
- `src/metrics.py`: 13 metrics (including multiclass Brier, ECE, and DW diagnostics).
- `experiments/`: experiment runner, statistical tests, and figure generation.

## Reproduce in 5 Commands

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python -m pytest tests/
python experiments/run_experiment.py --quick
```

Full pipeline:

```bash
python experiments/run_experiment.py --no-cv-lambda
python experiments/run_experiment.py
python experiments/statistical_tests.py
python experiments/visualize.py
```

## Notes

- `--quick` runs `{iris, wine, breast_cancer}` for smoke testing.
- `results/raw/all_folds.csv` is the main long-format fold-level output.
- `results/REPORT.md` is generated by `experiments/statistical_tests.py`.
- OpenML datasets are cached under `data/cache/`.
